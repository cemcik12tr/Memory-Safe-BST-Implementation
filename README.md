# Memory-Safe-BST-Implementation
TR:
Bu proje, C++ dilinde bellek  için güvenli bir ikili arama ağacı uygulamasını göstermektedir.
Program sonlanmadan önce kendi kendini temizleyerek hafızayı özgür bırakır, memory leak sorunun önüne geçilmeye çalışılmıştır.
ENG:
This project demonstrates how to run a memory-safe Binary Search Tree in C++.
It frees up memory by self-cleaning before the program terminates, helping to prevent memory leaks.
